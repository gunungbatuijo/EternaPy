EternaPy – Interactive Programming Learning Platform

Create a REAL, fully functional web application called EternaPy, inspired by Boot.dev, but with its own identity and learning experience.

Core Vision

EternaPy is a modern online platform where users can learn programming and software engineering skills through interactive lessons, coding challenges, projects, and AI-assisted guidance.

The design must feel professional, educational, and realistic.

Design Requirements

* No heavy neon effects.
* No cyberpunk style.
* Use a modern, professional learning-platform design.
* Inspired by platforms like:
    * Boot.dev
    * Coursera
    * Codecademy
    * Udemy
    * Khan Academy
* Clean typography.
* Soft shadows.
* Rounded corners.
* Professional color palette.
* Responsive for desktop, tablet, and mobile.
* Smooth animations.
* Dark mode and light mode support.

Eternal AI

Include an AI assistant called:

Eternal AI

Important Rule:

Eternal AI MUST NOT solve assignments or coding challenges directly.

Instead, it should:

Allowed

* Give hints
* Explain concepts
* Explain error messages
* Suggest debugging approaches
* Explain documentation
* Recommend learning resources

Not Allowed

* Generate final assignment solutions
* Complete coding challenges for users
* Reveal answers
* Auto-finish projects

If users ask for answers, Eternal AI should:

* Provide hints
* Ask guiding questions
* Explain relevant concepts

The goal is learning, not cheating.
